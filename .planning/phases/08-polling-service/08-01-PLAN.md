```python
class WorktreeHealthChecker:
    def __init__(self, orchestrator, check_interval=30.0) -> None:
        self._task: asyncio.Task[None] | None = None
    async def start(self) -> None:
        self._task = asyncio.create_task(self._check_loop())
    async def stop(self) -> None:
        if self._task:
            self._task.cancel()
            with contextlib.suppress(asyncio.CancelledError):
                await self._task
```

    **Constructor** (`__init__`):

    - Internal state: `_task: asyncio.Task[None] | None = None`, `_next_poll: dict[str, float] = {}` (profile name -> monotonic timestamp), `_active_tasks: set[asyncio.Task[None]] = set()` (for fire-and-forget tracking), `_repo_slugs: dict[str, str] = {}` (cache repo slugs per repo_root)

    **start()/stop()**: Identical to WorktreeHealthChecker pattern. `start()` creates task via `asyncio.create_task(self._poll_loop())`. `stop()` cancels task, suppresses CancelledError, also cancels all `_active_tasks`.

    **_poll_loop()**: While True: check `pr_polling_enabled` via `settings_repo.get_server_settings()`. If enabled, call `_poll_all_profiles()`. If disabled, skip silently. Sleep `tick_interval` between iterations. Wrap body in try/except Exception (log error, continue).

    **_poll_all_profiles()**: First check rate limit via `_should_back_off()` using the repo_root of the first enabled profile. If backing off, emit PR_POLL_RATE_LIMITED event, sleep until reset, return. Then iterate profiles from `profile_repo.list_profiles()`. Skip profiles where `pr_autofix is None`. Skip profiles whose `now < self._next_poll.get(name, 0)`. For each eligible profile, set `self._next_poll[name] = now + profile.pr_autofix.poll_interval` BEFORE calling `_poll_profile()` (prevents overlap). Wrap per-profile work in try/except Exception.

For each PR, call `fetch_review_comments(pr.number, ignore_authors=config.ignore_authors)`. If no comments, skip silently. If comments exist, get repo slug (cached), create fire-and-forget task for `orchestrator.trigger_fix_cycle(pr_number, repo, profile, head_branch, config)`. Track task in `_active_tasks` with done callback to discard. Log cycle summary at INFO if `triggered > 0 or len(prs) > 0`.
