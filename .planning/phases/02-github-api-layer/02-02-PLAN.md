**Class: `GitOperations`**
```python
class GitOperations:
    """Async git operations with safety guards for PR auto-fix.

    Uses create_subprocess_exec (not shell) for safety.
    """

    def __init__(self, repo_path: str | Path):
        self._repo_path = Path(repo_path)
```

**Private helper: `_run_git(self, *args: str, check: bool = True, timeout: float = 60.0) -> str`**
- Uses `asyncio.create_subprocess_exec("git", *args, stdout=PIPE, stderr=PIPE, cwd=self._repo_path)`
- `asyncio.wait_for(proc.communicate(), timeout=timeout)` with TimeoutError handling (kill process, raise ValueError)
- Non-zero exit + check=True: raise `ValueError(f"git {' '.join(args)} failed: {stderr}")`
- Returns stdout.decode().strip()

**Method: `stage_and_commit(self, message: str) -> str`** (GIT-01)
- `await self._run_git("add", "-A")`
- `await self._run_git("commit", "-m", message)`
- `sha = await self._run_git("rev-parse", "HEAD")`
- Return sha
- Log with loguru: `logger.info("Committed changes", sha=sha[:8], message=message)`

**Method: `safe_push(self, branch: str) -> str`** (GIT-02, GIT-03, GIT-04)
- Guard: `if branch in PROTECTED_BRANCHES: raise ValueError(f"Refusing to push to protected branch: {branch}")`
- `await self._run_git("fetch", "origin", branch, check=False)` -- don't fail if branch doesn't exist on remote
- `local_sha = (await self._run_git("rev-parse", "HEAD")).strip()`
- Try to get remote SHA: `remote_sha = await self._run_git("rev-parse", f"origin/{branch}")`. If ValueError/RuntimeError, set `remote_sha = None` (new branch).
- If remote_sha exists and != local_sha:
  - `merge_base = (await self._run_git("merge-base", local_sha, remote_sha)).strip()`
  - If `merge_base != remote_sha`: raise ValueError with divergence message (abort, never rebase per user decision)
- `await self._run_git("push", "origin", "HEAD")`
- Log: `logger.info("Pushed to remote", branch=branch, sha=local_sha[:8])`
- Return local_sha
