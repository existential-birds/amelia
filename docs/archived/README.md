## Why Archive?

Documents are archived when:
- **Superseded** - A newer design replaces this approach
- **Completed** - The work is done and merged
- **Abandoned** - The direction was not pursued
- **Historical** - Useful for understanding past decisions

## Policy

Archived documents are **read-only reference material**. Do not update them. If an archived approach becomes relevant again, create a new document in the appropriate folder and reference the archived version.
